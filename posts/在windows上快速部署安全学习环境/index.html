Shell与终端#

除开个别跨平台的shell, 以及git bash等shell, powershell 7 是windows下shell最好的选择. 注意powershell 7不同于windows自带的powershell, 两者是不同的powershell版本. 安装方法参见该 链接

windows terminal是目前windows 终端的最佳选择, 你也可以尝试其他跨平台的终端, 例如tabby.

windows包安装管理工具: Scoop#

scoop(可能)是windows上的最好用的包安装管理工具.

scoop官网提供的默认的安装方式会将scoop和以后所有scoop安装的软件放在C盘, 考虑到C盘的容量, 务必使用 ScoopInstaller/Install所述的Advanced Installation方法进行安装.

在安装scoop后, 必须设置代理才能获得流畅体验. 参见scoop wiki: Using-Scoop-behind-a-proxy

给scoop添加bucket, 这样可以下载更多软件(后面安装工具所必须)

scoop bucket add extras
scoop bucket add java

常用工具下载#

一些常用的工具, 按需安装.

scoop install main/7zip # 最好用的压缩包软件
scoop install main/git # 非常常用
scoop install extras/googlechrome # 你总需要一个浏览器的, 对吧?!
scoop install extras/sumatrapdf # windows下最好用的pdf阅读器
scoop install extras/autohotkey # 也许有一天你会需要他的 :)

开发环境配置#

按需

scoop install main/gcc
scoop install extras/miniconda3 # python env
scoop install extras/vscode # editor

VMware Workstation Pro#

VMware Workstation Pro, 最好用的虚拟机软件.

VMware Workstation Pro现在下载需要登陆, 免登录下载的方法可以参考 https://www.cnblogs.com/dianchaozhang/p/17268234.html.

或使用此链接: VMware Workstation Pro 17.0.2

密钥可用: MC60H-DWHD5-H80U9-6V85M-8280D.

如果你不想通过镜像文件手动安装获取虚拟机, 那么 OSBoxes是你的不二之选, 它提供了许多下载即用的虚拟机

kali官方提供了kali virtual machines.

BurpSuite Pro 安装(2023.9可用)#

BurpSuite Pro官网下载官方的安装包(点击download), 然后运行该安装包.

或者, 使用scoop

scoop bucket add nonportable
# scoop install nonportable/burp-suite-community-np # 社区版
scoop install nonportable/burp-suite-pro-np # pro版本

在https://github.com/x-Ai/BurpSuite下载BurpSuiteLoader.jar, 并将该文件放入BurpSuite Pro安装目录(即, burpsuite_pro.jar 所在目录)

在爱盘 下载 keygen文件, 解压获取burp-loader-keygen.jar文件, 同样放入放入BurpSuite Pro安装目录.

依照https://github.com/x-Ai/BurpSuite的指示, 在Burpsuite Pro 安装目录下, 使用命令行执行下面的命令.

java -noverify -Dsun.java2d.d3d=false -Dsun.java2d.noddraw=true --add-opens=java.base/jdk.internal.org.objectweb.asm=ALL-UNNAMED --add-opens=java.base/jdk.internal.org.objectweb.asm.tree=ALL-UNNAMED -javaagent:BurpSuiteLoader.jar -jar burpsuite_pro.jar

如果上面的命令报错, 则根据报错信息, 删除对应的选项, 例如执行

java -noverify --add-opens=java.base/jdk.internal.org.objectweb.asm=ALL-UNNAMED --add-opens=java.base/jdk.internal.org.objectweb.asm.tree=ALL-UNNAMED -javaagent:BurpSuiteLoader.jar -jar burpsuite_pro.jar

如果命令执行成功, 此时burp应当已经启动(记住这条命令, 以后我们每次启动burp都必须使用这条命令), 阅读LICENSE后, 会被要求输入key.

此时再新开一个终端, 同样实在安装目录下运行如下面命令:

java -jar burp-loader-keygen.jar

这会启动keygen, 帮助我们完成注册.

  • 首先, 你可以自定义一下 License Text.
  • Copy License中的内容到Burp中, 然后点击next, 选择Manual activation.
  • 将此时Burp提供的Request部分的内容copy(点击Copy request即可)到 keygen的 Acitvation Request区域
  • 将此时keygen计算出Activation Response区域的内容paste到Brup的对应区域(点击Paste response)
  • 点击 next, 注册成功

由于每次启动都需要手动输入过长的命令, 这很不方便. windows下, 我们可以在burp安装目录下创建StartBrup.vbs文件, 内容如下:

Set WshShell = WScript.CreateObject("WScript.Shell")
WshShell.Run "java -noverify --add-opens=java.base/jdk.internal.org.objectweb.asm=ALL-UNNAMED --add-opens=java.base/jdk.internal.org.objectweb.asm.tree=ALL-UNNAMED -javaagent:BurpSuiteLoader.jar -jar burpsuite_pro.jar", 0

以后只需要双击该vbs文件即可启动burp, 使用该脚本同时也隐藏了cmd, 更加美观.

注意: 必须使用上述vbs文件启动或者在cmd使用命令启动burp, 如果使用burp默认的启动方式会显示未注册.

逆向工具安装#

IDA pro 爱盘下载

JEB Pro/JEB Android 爱盘下载

scoop可安装:

# 二进制文件编辑器
scoop install extras/010editor # 破解补丁 https://down.52pojie.cn/Tools/Editors/010_Editor_All_Versions_For_Windows_CracKed.7z

# PE文件工具
scoop install extras/exeinfo-pe
scoop install extras/detect-it-easy
scoop install extras/resource-hacker

# 调试器
scoop install extras/ollydbg # OD已经够老了, 看雪或者52破解提供的OD更好用 # https://down.52pojie.cn/Tools/Debuggers/吾爱破解专用版Ollydbg.rar
scoop install extras/x64dbg
scoop install extras/dnspy # .NET debugger and assembly editor

scoop install extras/ilspy # .NET assembly browser and decompiler.

scoop install java/oraclejdk-lts # java软件运行必须, 重启后生效.
scoop install extras/jd-gui
scoop install extras/bytecode-viewer

scoop install extras/wireshark

如果你要用过时的PEid, 选择看雪版本的PEid, 你还可以在同一页面下载看雪版本的ExeinfoPE(选下载次数最多的那个, 与scoop安装的相比, 看雪版附带很多东西)

GDA也是一个优秀的工具, 不过需要手动下载.
